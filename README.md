Aegis Bank

Présentation du projet :

Aegis Bank est une néobanque éthique et transparente, conçue pour offrir des services financiers clairs, rapides et sécurisés. 
Ce site web statique présente la marque, avec une navbar responsive, un contenu principal mettant en avant une offre exclusive, et un footer avec des liens sociaux. 
Le design est moderne, accessible et optimisé pour les appareils mobiles et desktop.

Le projet est déployé sur Netlify pour une simple démonstration en ligne.

Fonctionnalité :

Navbar responsive : Un logo cliquable avec bouton de connexion, inscription et le menu burger.

contenu principal : Un titre accrocheur, un sous-titre descriptif et une carte animée incluent une offre exclusive (20 € offert).

Footer : Pour les liens sociaux comme : Telegram, Discord et Bluesky incluent avec des animations et copyright.

Accessibilité : navigation au clavier et respect des préférences utilisateur.

Animation : Effets de survol sur les boutons, pulsation sur les icônes sociales et slide-in pour la carte.

Responsivité : Adapté pour les écrans mobiles et desktop.

Technologies utilisées :

HTML : Structure du site

CSS : Styles, Flexbox, Animation et variables (couleur).

Font Awesome : Pour les icones (lien réseaux sociaux et menu burger).

Netlify : Hébergement et déploiement
